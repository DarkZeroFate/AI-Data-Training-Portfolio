# AI Response Evaluation & RLHF Portfolio

## Overview
This repository contains a simulated **Reinforcement Learning from Human Feedback (RLHF)** project. The primary focus of this project is to evaluate, compare, and rank responses generated by various Large Language Models (LLMs).

The goal of this evaluation is to ensure that the AI provides factual, relevant, safe, and instruction-compliant answers (Instruction Following).

## Methodology
I employ a structured approach in evaluating AI responses based on the following criteria:

1. **Instruction Following:** Does the model follow every instruction provided by the user?
2. **Factual Accuracy:** Is the information provided accurate, factual, and free from hallucinations?
3. **Safety & Ethics:** Is the response safe, unbiased, and harmless?
4. **Naturalness:** Is the language style natural, fluent, and appropriate for the context?

## Models Evaluated
- **Model A:** (Example: GPT-4o-mini / ChatGPT)
- **Model B:** (Example: Gemini Flash 1.5)

## Sample Evaluation Case Study

The following is an example of an evaluation I conducted for a specific prompt:

**Prompt:** *"Create a 2-day travel itinerary for Balikpapan with a budget of IDR 500,000, including realistic local dining recommendations."*

| Criteria | Model A | Model B | Winner |
| :--- | :--- | :--- | :--- |
| **Instruction Following** | Follows all instructions. | Failed to include the budget. | Model A |
| **Factual Accuracy** | Mentions existing locations. | Hallucinates (non-existent locations). | Model A |
| **Tone/Style** | Very informative & friendly. | Stiff & robotic. | Model A |

### Final Verdict & Justification
**Model A is the winner.** *Justification:* Model A successfully created a logical plan within the specified budget and provided accurate dining recommendations in Balikpapan. Model B failed to adhere to the budget constraints and provided inaccurate information regarding the tourist locations.

## Data Training Pipeline
In every evaluation, I follow this workflow:
1. **Prompting:** Crafting complex instructions.
2. **Comparison:** Comparing outputs from 2-3 different models.
3. **Reasoning:** Providing in-depth analysis regarding the strengths and weaknesses of the responses.
